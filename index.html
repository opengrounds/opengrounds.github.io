what is open:grounds?

a constellation of maps for the things we share.

open:grounds started with one question — what's free, public, and shared in davis, ca? the answer became a map. everything else here grows from that same idea: common goods are easier to find when someone draws the map.

each house on the map is a project. filled houses are live. outlined ones are still being built — square by square, the same way you'd build a house.
